# 🎓 Remote Classroom for Rural Colleges (SIH 2025 Project)

A **Smart Education** platform built for **Smart India Hackathon (SIH) 2025**.  
This project aims to provide a **Remote Classroom solution for rural colleges**, enabling students and teachers to connect seamlessly for live classes, assignments, and resources.

---

## 🚀 Problem Statement
**PS Number:** SIH25101  
**Theme:** Smart Education  
**Title:** Remote Classroom for Rural Colleges  
**Category:** Software  

Rural students often face challenges accessing quality education due to distance, limited infrastructure, and lack of digital resources.  
Our solution bridges this gap by creating a **digital classroom platform** accessible via mobile and web, supporting **live teaching, learning resources, and interactive tools**.

---

## ✨ Features
✨ Features (Update)
👨‍🏫 Role-based Dashboards (Student & Teacher)
🎥 Live Classroom (Video + Chat + Attendees)
📂 Assignments & Resources (Upload + Submit)
🔔 Notifications (Deadlines, Alerts)
🔑 OTP-based Authentication
🌐 Multi-Language Ready (English + Hindi)
🤖 AI Chatbot – student queries ke liye (ChatGPT-like support)
🎯 AI Recommendation System – personalized class & resource suggestions

---

## 🛠 Tech Stack
Frontend: React.js, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB (Mongoose ORM)
Authentication: JWT + OTP Verification
Real-Time: WebRTC (Video), Socket.IO (Chat)
AI/ML: NLP-based Chatbot (Dialogflow / custom ML), Recommendation Engine (Content + Collaborative filtering), Python APIs
Other Tools: Figma (UI/UX), GitHub, Render, Netlify 

---

## 📊 System Flow
1. **Landing Page** → Choose role (Student / Teacher)  
2. **Signup/Login** → OTP-based authentication  
3. **Student Dashboard** → Upcoming Classes, Assignments, Resources  
4. **Teacher Dashboard** → Create Classes, Upload Materials, Manage Students  
5. **Classroom Page** → Live video, chat, attendees  
6. **Profile/Settings** → Manage account
